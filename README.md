## FIEL CONVERSOR 

O Fiel Conversor é um programa que permite converter imagens em PDF, PDF em imagens, extrair páginas selecionadas de um PDF, agrupar arquivos em PDF em um só, girar páginas, e converter imagens de texto para PDF com OCR, através do Tesseract suportando diversas línguas. Com esse programa, você pode:

   - Converter imagens nos formatos JPG, PNG, BMP, GIF e TIFF em arquivos PDF, com a possibilidade de escolher a quantidade de imagens a serem adicionadas;

   - Converter arquivos PDF em imagens nos formatos JPG, PNG, BMP, GIF e TIFF, podendo escolher a resolução das imagens;

   - Extrair páginas selecionadas de um arquivo PDF, criando um novo arquivo PDF com as páginas desejadas, bastando informar o intervalo de páginas desejado;

   - Agrupar vários arquivos em PDF em um só, criando um novo arquivo PDF com todas as páginas dos arquivos originais;

   - Girar as páginas informadas de um PDF selecionado, podendo escolher o ângulo de rotação da página (90, 180 ou 270 graus); e

   - Converter imagens de texto para PDF com OCR, reconhecendo o texto nas imagens, criando um arquivo PDF pesquisável e editável. Você pode usar o OCR em imagens nos formatos JPG, PNG, BMP, GIF e TIFF, e escolher o idioma (14 idiomas disponíveis!) para aplicar no texto das imagens.

Além disso tudo, é possível ainda aplicar OCR em todo um arquivo PDF com inúmeras páginas. Para isso, primeiro converta o PDF que deseja passar OCR em imagens com o "Conversor de PDF para Imagens", depois converta as imagens para PDF com OCR através do "Conversor de Imagens de Texto para PDFs pesquisáveis (OCR)" e, por fim, agrupe todos os PDFs com OCR aplicados com o "Agrupador de Arquivos em PDF". Pronto! Agora você tem seu arquivo em PDF todo pesquisável com OCR aplicado.

Obs.: note que se for transformar as páginas de um PDF em imagens para depois aplicar o OCR nessas imagens, é desejável que elas estejam com uma qualidade adequada para que se tenha uma experiência equilibrada entre qualidade do OCR e tamanho do arquivo. Para textos com fontes pequenas é recomendável uma quantidade maior do que os 72 DPIs que estão selecionados por padrão na seção "Conversor de PDF para Imagens". Na dúvida selecione 150 DPI. Mas perceba que tal decisão afeta o tamanho dos arquivos de imagem (ocupa mais espaço de armazenamento) e o tamanho dos arquivos de imagem afeta o tempo de conclusão da tarefa (demora mais tempo para converter tudo em PDFs com OCR aplicado).

O Fiel Conversor é um programa fácil de usar, rápido e eficiente, que pode ajudá-lo nas necessidades de conversão de arquivos no dia a dia de forma rápida, prática e segura.

### **Método Recomendado: Executável Pré-Compilado (mais fácil)**

Não é necessário instalar Python nem rodar `pip`. Basta baixar o pacote binário correspondente ao seu sistema.

#### Passo 1: Baixar o executável pronto
Baixe diretamente do repositório o arquivo da última versão:

- **Windows (10/11 de 64 bits)**: `Fiel_Conv-v.x.x.x-bin-Windows-x64.zip`
- **Linux (Ubuntu/Mint/Debian de 64 bits)**: `Fiel_Conv-v.x.x.x-bin-Ubuntu-x64.tar.xz`

#### Passo 2: Extrair, criar a estrutura de pastas e rodar o programa
1. Extraia o arquivo baixado para uma pasta chamada **`Fiel_Conversor`** (pode ser qualquer nome, mas recomendo esse).
2. Dentro da pasta `Fiel_Conversor` você já terá:
   - `Fiel_Conv-v.x.x.x.exe` (Windows) ou `Fiel_Conv-v.x.x.x` (Linux)
   - Pasta `_internal`
3. Como rodar o programa:
   - **Windows**: dê duplo clique em `Fiel_Conv-v.x.x.x.exe`
   - **Linux**: 
     ```bash
     chmod +x Fiel_Conv-v.x.x.x
     ./Fiel_Conv-v.x.x.x
     ```

A janela GUI vai abrir automaticamente.

### **Método Alternativo: Instalar via Código Python (Fiel_Conv-v.x.x.x.py)**

Caso prefira rodar diretamente do fonte (ex.: para desenvolvimento ou outras versões do Python):

1. Tenha **Python 3.13** instalado.
2. Rode:
   ```bash
   pip install PyPDF2 pymupdf pillow
   ```
3. Crie uma pasta Tesseract e copie nela os binários e executáveis do tesseract.
4. Coloque o arquivo `Fiel_Conv-v.x.x.x.py` na raiz da pasta `Fiel_Conversor`.
5. Rode: `Fiel_Conv-v.x.x.x.py`
